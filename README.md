# Lift – Workout Tracker

A personal PWA workout tracker built for daily use on iPhone.

## Features
- Log workouts by muscle group
- Track weight over time
- Personal records (PRs)
- Badges and streaks
- Backup & restore (export/import JSON)
- Works offline

## Install on iPhone
1. Open the app URL in Safari
2. Tap Share → "Add to Home Screen"
3. Launches fullscreen like a native app

## Data & Privacy
All data is stored locally on your device (`localStorage`). Nothing is sent to any server.

## Tech
Plain HTML, CSS, JavaScript. No frameworks, no dependencies. Hosted on GitHub Pages.
